# CRM / Client Management Dashboard UI

## 📌 Project Overview
This project is a **CRM (Client Relationship Management) / Client Management Dashboard UI** designed for a small agency or B2B service-based business.  
The goal is to create a clean, efficient, and user-friendly system to manage leads, clients, tasks, and follow-ups in one place.

---

## 🎯 Objective
To improve workflow efficiency by designing a dashboard that helps agencies:
- Track leads and conversions
- Manage client relationships
- Organize tasks and follow-ups
- Visualize pipeline status clearly

---

## 🧩 Key Features

### 1. Lead Tracking & Client List
- Centralized database of all leads and clients
- Search and filter options
- Status-based categorization (New, In Progress, Converted, Lost)

### 2. Visual Lead Pipeline
- Drag-and-drop style pipeline view
- Stages like:
  - Lead Generated
  - Contacted
  - Proposal Sent
  - Negotiation
  - Converted

### 3. Client Profile View
- Detailed client information
- Communication history
- Engagement status tracking

### 4. Task Tracker & Follow-ups
- Task assignment system
- Due dates and reminders
- Priority labeling

---

## 🎨 Design Tools Used
- Figma (Primary UI/UX Design Tool)
- Adobe XD / Framer (Optional for prototyping)

---

## 🧠 Design Thinking & Rationale
The design focuses on:
- **Clarity:** Simple layout to avoid cognitive overload
- **Efficiency:** Quick access to leads and tasks
- **Workflow Optimization:** Matches real agency operations
- **Data Visibility:** Clear pipeline and status tracking

The structure is inspired by real-world SaaS CRM tools used in agencies and startups.
